COSC2196 Introduction to Information Technology Assessment 1: My Profile

Runeterra Informative / Guide Website

Project Idea

Overview

The following report will be about a project idea about creating a website based around the game called Runeterra which is created by Riot. This is a strategy card game and when I played it felt as if there weren’t any content online to guide the players on how to play and strategies to win.

The website will have three main sections:

  • How-to Play Basics
  • Strategies
  • Top Tier Decks

I will also try to implement a news section which will pull its data straight from the official website, which will show upcoming updates and news that the community will be interested in.

Motivation

This project will be a great vessel for all types of players who are just looking to improve their skillset or find a different play style to change things up. It will be interesting and beneficial for those who use it, allowing them to have an advantage over their competitors by using the strategies and guides provided on the website. Building this website will also help me learn the tools I require to apply or compete for my ideal job for a Front-End Web Developer.

Features of the Service

The website will be broken in three sections as mentioned in the overview.

  • How-to Play Basics: The How-To Play basics of the website will inform the users about the basics of the game, such as describing basic concepts, how cards interact with each other and the best ways to visualise/strategise each scenario based on the cards you have been given.
  • Strategies: This will be detailed descriptions about different strategies the players can try based off the decks they are using, and how each card can affect play.
  • Top Tier Decks: This section will be describing the best decks currently in season that are winning consistently and played by top ranking players.

I will also integrate user friendly navigation; this will allow the users who use the site to easily navigate around the page and keep them coming back. The less they need to think about whilst using the site the better.

Tools and Technologies / Skills Required

The software required to get this project up and running are as follows:

  • Domain Name: Using such providers as GoDaddy or Domain.com, this service should only set you back $20 a year depending on the name unless it’s a premium name then they can go up to hundreds or thousands of dollars
  • Text Editor:Sublime Text / Notepad++ / Visual Studio Code – This is what will be required to build up the website’s pages, using HTML, CSS and JavaScript maybe jQuery if need be
  • Web Browsers:To check the work prior to building the website such as Chrome, Firefox or IE, also allowing you to access developer tools
  • Version Control Tools:Github – This will make sure you have versioning control and can rollback or upgrade when need be
  • Visual Server: A host website or service such as Amazon AWS or Google Cloud Platform
  • Image Manipulation Software: Adobe Photoshop – This will help editing images to make it perfect for what is intended for the site and adjust aspect ratios
  • Bootstrap:This tool is very handy if you want to use this to form a template using HTML, CSS and Javascript to create the base of your website

Using the tools above I will be able to create the website required for the project to the best of my ability.

The skills are easily obtainable simply by putting in the time to learn the programs/software above as they are mostly all open source. The trickiest part would be to sync up all the data across multiple platforms, to make it as widely accessible as possible, providing access on any given device.

Outcome

If this project is created and is successful, this will impact thousands if not millions of the player-base to be able to research new in-game strategies and guides. Having this website will also keep the player-base consistently interested in the game by creating new strategies for them to try and accomplish. New players will also find it informative as this is mostly targeted towards the new player-base. This will be a one stop shop (website) for all players to go to for anything related to Runeterra as the news section of the website will be directly linked to the official website.
